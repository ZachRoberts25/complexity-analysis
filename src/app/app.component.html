Explanation of Data

The following graphs and statistics were calculated by running projects through a complexity analysis using a javascript library called escomplex. The output of the analysis is a report including Halstead Complexity Measures, Sloc, and Cyclomatic Complexity. The report was then aggregated to show the cool statistics you see below.
